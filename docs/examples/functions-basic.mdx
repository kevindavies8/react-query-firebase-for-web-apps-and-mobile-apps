# Functions: Basic Example

This example loads a joke from a deployed [callable HTTPS function](https://github.com/invertase/react-query-firebase/blob/main/functions/src/index.js) called `getJoke`, using the the
`useFunctionsQuery` hook, and allows you to fetch a new joke using the `useFunctionsMutation` hook.
